# Chapter 12: Building Your Own Token

To install the dependencies of the project, run the following command:
```bash
npm install
```

##### Compile
```bash
truffle compile
```

##### Run migration scripts
```$bash
truffle migrate
``` 

##### Run test cases
```$bash
truffle test
```

## On Testnet using Rinkeby

##### Deploy
```bash
truffle migrate --network rinkeby
```
